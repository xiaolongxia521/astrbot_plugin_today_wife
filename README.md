# astrbot_plugin_today_wife
抽老婆插件，并且只能抽中当天发言过的人，而且不会出现一个人是两个人的老婆的情况，会两两绑定


📖 简介
这是一个为 AstrBot QQ 机器人框架设计的趣味插件，实现"今日老婆"功能。插件会记录当天在群内发言的用户，允许用户随机抽取一位"老婆"，并且一旦配对成功，双方都会被绑定，不能再被其他人抽取。

✨ 功能特点
智能记录：自动记录当天在群内发言的用户（排除命令消息）

随机配对：使用 /今日老婆 命令随机抽取一位当天发言的群友

排他性绑定：一旦配对成功，双方都会被锁定，不能再被其他人抽取

每日重置：每天凌晨自动清空配对关系，重新开始新一天

多群支持：支持多个群聊独立运行，互不干扰

高性能：异步架构设计，不影响机器人正常响应
